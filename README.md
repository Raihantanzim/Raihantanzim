### SafeStreets-LDN
**Open-Data Road Safety Risk Mapping with GNNs**

Built a graph-based crash-risk prediction pipeline for London road intersections using **OpenStreetMap road networks** and **STATS19 collision data**. Engineered node and edge geospatial features, trained **TransformerConv GNNs**, and compared results against **XGBoost** and baseline models using spatial holdout evaluation.

**Stack:** Python, PyTorch Geometric, GeoPandas, OSMnx, XGBoost

**End-to-end ML pipeline for delivery operations**

**Stack:** Python, scikit-learn, pandas, matplotlib
